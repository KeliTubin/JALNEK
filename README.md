1 Nodemon install

```
yarn global add nodemon
```

Nodemon run

```
nodemon
```

2 Typescript install

```
yarn global add typescript
```

3 tsc install

```
yarn tsc --init
```

4 ts-node install

```
yarn add  ts-node -dev
```

5 Node types

```
yarn add  @types/node
```

6 START SERVER

```
yarn start
```
